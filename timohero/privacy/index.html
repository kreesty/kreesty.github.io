Personal Data

Timo Hero is committed to protecting your privacy. It does not collect or save any personal information or browsing history.

If there's ever a need to store data—for instance, to support a new feature—it will only happen with your explicit consent.

Third-Party Services

Timo Hero uses Chrome's Storage Sync API to securely save user login details and Chrome's Storage Local API to store settings like email templates.

For managing payments, Timo Hero relies on ExtPay and Stripe.
